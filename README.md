# brand — Coming Soon T-shirt store

India-first ecommerce storefront for **brand** (placeholder name). Built with Next.js, Tailwind CSS, and a client-side cart.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the port Next prints if 3000 is busy).

## What's included

- Home, Shop, Product (with size/colour/logo customization), Cart, Checkout
- About, Contact, Size guide, FAQ, Shipping, Privacy, Terms
- Responsive nav + cart drawer
- INR pricing, COD/UPI/card checkout demo, pan-India copy
- Coming Soon early-access positioning (tees only)

Checkout is a front-end demo — orders confirm locally and are not charged.
